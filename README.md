# Projets académiques – Génie informatique

Ce dépôt regroupe une sélection de projets académiques réalisés dans le cadre du
baccalauréat en génie informatique à l’Université du Québec à Chicoutimi (UQAC).

Les projets présentés illustrent mes compétences en programmation,
en conception de systèmes et en bases de données.

Ces travaux ont été réalisés dans un contexte académique.
